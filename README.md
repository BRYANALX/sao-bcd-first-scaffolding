# sao-bcd-first-scaffolding

> Primer generador con Sao.Js

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao bcd-first-scaffolding my-project
```

### From git

```bash
sao bryanalx/sao-bcd-first-scaffolding my-project
```
